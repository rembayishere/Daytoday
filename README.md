# Daytoday

> 灵感来了随手记 · 知识沉淀日日新

**Daytoday** 是一款轻量、优雅的本地优先（local-first）知识管理桌面应用，帮你把散落的灵感、待解的问题、要背的卡片和手头的任务——统统收进一个窗口。数据默认存在本地，可经 **WebDAV + AES-256-GCM** 加密同步到你自己的私有云盘，真正「数据由你掌控」。

## 🖥️ 在线互动预览

不想安装也能完整体验界面与交互？项目内置了一个 **可交互的 HTML 原型**，直接用浏览器打开即可点击切换各个面板、浏览热力图、添加闪卡与问题：

```bash
# 克隆后直接用浏览器打开
open demo.html        # macOS
# 或 Windows 下双击 demo.html
```

原型包含全部页面：笔记、智能总结、剪藏、问题追踪、闪记卡片、任务看板、设置，以及可翻月切换的活跃度热力图。

## ✨ 功能一览

| 功能 | 说明 |
| --- | --- |
| 📝 **笔记** | 像 Flomo 一样畅快书写，支持 `#标签` 与日期筛选，`Ctrl+Enter` 发送，`Alt+Q` 唤起快速记录弹窗。 |
| 🧠 **智能总结** | 按标签自动聚合笔记，生成内容卡片与统计，写得越多总结越有营养。 |
| 🔖 **剪藏** | 保存文章标题与链接，沉淀你的个人阅读档案库。 |
| ❓ **问题追踪** | 记录问题并跟踪状态（待解决 → 进行中 → 已解答），附带自动版本历史，可回溯任意两次修改。 |
| 🃏 **闪记卡片** | 正面 / 反面卡片，点击翻转复习，支持标签分类。 |
| 📋 **任务看板** | 三栏看板（待办 / 进行中 / 已完成），拖拽改状态，高 / 中 / 低优先级标记。 |
| 📁 **资料库** | 本地保存任意文件，可拖拽上传、建文件夹分类，并**关联笔记**；支持随 WebDAV 一起加密上传。 |
| 🔥 **活跃度热力图** | 侧边栏日历热力图展示每日记录密度，点选任意日期即可筛选。 |
| ☁️ **WebDAV 同步** | AES-256-GCM 加密后同步至私有云盘，可选择同步范围与「软件设置同步」开关。 |
| 🤖 **AI 配置** | 兼容 OpenAI API 格式，可对接本地 LLM（Ollama、LM Studio）或云端服务。 |

### 📁 资料库 —— 你每天成果的归处

资料库用来沉淀你产出的各种文件（文档、图片、压缩包、导出物……），**默认全部保存在本机**，但和笔记是打通的：

- **本地优先**：文件存于本机资料目录，不上传也完全可用。
- **拖拽即传**：把文件直接拖进窗口即可上传，支持多选与任意类型。
- **文件夹分类**：按项目 / 主题建文件夹，资料井井有条。
- **关联笔记**：每条资料都能「链接」到一条或多条笔记。比如你今天写了「完成 XX 报告」的笔记，就把报告 PDF 关联上去——日后翻笔记就能顺手找回那份文件。
- **随 WebDAV 上传**：开启同步后，资料会与笔记一起 AES-256-GCM 加密上传到你的私有云盘（需在同步设置中勾选「资料文件」）。

#### 推荐用法：每天记录 → 沉淀成果 → 随时找回

```
白天   写笔记：「今天搞定了客户方案初稿 #工作」
        ↓ 关联
       上传「方案初稿.pdf」到资料库，并链接上面那条笔记

晚上   打开资料库，按文件夹 / 关联笔记浏览当天成果
       开启 WebDAV 同步，资料与笔记一并加密备份到云端

日后   想回忆「某天干了啥」→ 看热力图点那天 → 笔记 + 资料一目了然
       想找「那份方案」→ 搜笔记关键词 → 直接跳转关联的文件
```

资料库让「记录」和「产出」绑定在一起：不只是记下了做了什么，连做出来东西也留住了，复习与检索都更省力。

## 🛠️ 技术栈

| 层 | 选型 |
| --- | --- |
| 界面 | React 19 + TypeScript |
| 构建 | Vite 8 |
| 状态 | Zustand 5 |
| 桌面壳 | Tauri 2（Rust 后端） |
| 加密 | AES-256-GCM（argon2 派生密钥） |
| 同步 | WebDAV（reqwest） |
| 存储 | 本地 JSON，可选 WebDAV + AES 加密同步 |

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式（前端热更新 + Rust 后端）
npm run tauri dev

# 打包分发（生成 Windows MSI）
npm run tauri build
```

构建产物位于：

```
src-tauri/target/release/bundle/msi/Daytoday_1.0.0_x64_zh-CN.msi
```

### 安装包说明

- 当前提供 **Windows x64 MSI** 安装包。
- 首次启动即创建本地数据目录，所有笔记默认只存在本机。
- 需在「设置 → WebDAV 同步」中填写你自己的 WebDAV 地址与**专用应用密码**后，才会进行加密同步。

---

## ☁️ WebDAV 同步配置要点

1. 在网盘服务商处生成 **WebDAV 专用应用密码**（注意：通常是独立授权码，不是登录密码）。
2. 服务器地址按需填写，例如 123 云盘：`https://webdav-<你的UID>.pd1.123pan.cn/webdav`。
3. 在「服务器连接」分组填账号密码，在「数据加密」分组设置加密密码（留空则不加密）。
4. 在「软件设置同步」分组可开关是否把 AI 配置、快捷键等设置一起加密同步。
