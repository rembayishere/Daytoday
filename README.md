# Daytoday

> 灵感来了随手记 · 知识沉淀日日新

**Daytoday** 是一款轻量、优雅的本地知识管理桌面应用，帮你把散落的灵感、待解的问题、要背的卡片和手头的任务——统统收进一个窗口。

---

## 它能做什么

### 📝 笔记 — 像 Flomo 一样畅快书写
支持 #标签 和日期筛选，**Ctrl+Enter** 快速发送，**Alt+Space** 全局唤起快速记录，随时随地捕捉想法。

### 🧠 智能总结 — 笔记自己长成摘要
按标签自动聚合你的笔记，生成内容卡片和统计。写得越多，总结越有营养。

### 🔖 剪藏 — 网页碎片收集器
保存文章标题与链接，形成你的个人阅读档案库。

### ❓ 问题追踪 — 每个问题都有下文
记录问题，跟踪状态（待解决→进行中→已解答），附带**自动版本历史**，可以回溯每一次修改、比对任意两个版本。

### 🃏 闪记卡片 — 像 Anki 一样刷
正面/反面卡片，点击翻转，支持标签分类。做自己的记忆宫殿。

### 📋 任务看板 — 拖拽即管理
三栏看板（待办 / 进行中 / 已完成），拖拽改变状态，支持高/中/低优先级标记。

### 🔥 活跃度热力图 — 你的努力看得见
侧边栏热力图直观展示每日记录密度，点击任意日期即可筛选查看，让坚持可视化。

### ☁️ WebDAV 同步 — 数据由你掌控
AES-256-GCM 加密后同步至你的私有云盘（NextCloud、Box 等），数据永远属于你。

### 🤖 AI 配置 — 接入你喜欢的模型
兼容 OpenAI API 格式，可对接本地 LLM（Ollama、LM Studio）或云端服务。

## 技术栈

| 层 | 选型 |
|---|---|
| 界面 | React 19 + TypeScript 6 |
| 构建 | Vite 8 |
| 状态 | Zustand 5 |
| 桌面壳 | Tauri 2（Rust 后端） |
| 存储 | 本地 JSON（WebDAV + AES 加密同步） |

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 打包分发
npm run tauri build
```

构建产物位于 `src-tauri/target/release/bundle/msi/`。
